# Cloud Incident Review

## Incident summary

Checkout API depends on Redis Cluster. Checkout API also depends on Payment Token Service. Payment Token Service uses Key Management Platform. Redis Cluster is owned by Cloud Reliability Team.

## Ownership

Northstar Payments Team is responsible for Checkout API. Cloud Reliability Team supports Northstar Payments Team. Incident Recovery Program is sponsored by Chief Technology Officer David Kim. David Kim is responsible for Incident Recovery Program.

## Recovery work

Observability Guild supports Incident Recovery Program. Observability Guild partners with Cloud Reliability Team. Incident Recovery Program uses Recovery Scorecard. Marcus Lee reviews Recovery Scorecard. Recovery Scorecard reviews Checkout API.

## Outcome

The incident lasted forty seven minutes. The recovery target is to restore Checkout API within fifteen minutes and reduce repeated payment attempts by thirty percent.
